Qibla iFinder Support

How the App Works:

Upon opening the Qibla iFinder App the user is presented with the "Please Enable Location Services" request prompt. The "Allow Location Services" button enables the app to use Location Services to direct you towards the Kaaba. After pressing the "Allow Location Services" the user is presented with 3 options: "Allow Once", "Allow While Using App", "Don't Allow".

Please Select "Allow While Using App" For An Optimal User Experince

The "Allow Once" Option

By selecting the "Allow Once" option, Qibla iFinder will present the Qibla angle. Upon closing the app, "Please Enable Location Services" will once again be prompted.

The "Allow While Using App" Option

Once selecting the "Allow While Using App" option, the user will be pointed towards the Kaaba. The user will Not be prompted again. This option is recommended for optimal user experience.

The "Don't Allow" Option

This option will prompt an error message. Please go to your iOS device's Settings. Find and Select Qibla iFinder. Allow access to Location.
